url structure

The website has some code that takes the url and interprets the content in the middle frame from it. All of the content on the webpage is on the same page, it just fills in the center depending on what the url says.
Example:

Aeriecat.space/#p/weblog

it puts everything after the # in the iframe. (in this case, the file p/weblog) if no file extention is given, it assumes .html
